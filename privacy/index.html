Política de Privacidade

Esta política explica, de forma clara, quais dados podem ser usados pela AlphaX para ativação, suporte, segurança e funcionamento do aplicativo.

Resumo

A AlphaX não vende informações pessoais. Os dados são usados para operar o app, validar ativações, responder suporte e proteger o serviço.

7. Contato

alphaxapp@gmail.com

1. Escopo

Esta política se aplica ao site AlphaX, às páginas de suporte e ao uso do aplicativo AlphaX em dispositivos compatíveis.

2. Dados que podemos coletar

  • Identificadores técnicos do dispositivo usados para ativação.
  • Status de licença, data de ativação e informações necessárias para validar acesso.
  • Modelo do aparelho, versão do sistema e mensagens enviadas em solicitações de suporte.
  • Dados técnicos mínimos para diagnosticar erro, abuso, fraude ou acesso indevido.

3. Como usamos os dados

Usamos dados para ativar o aplicativo, responder chamados, corrigir falhas, melhorar estabilidade e proteger a operação. Não usamos dados de suporte para vender publicidade.

4. Retenção

Mantemos informações apenas pelo tempo necessário para suporte, operação, prevenção de abuso e obrigações legais aplicáveis.

5. Compartilhamento

Podemos usar provedores técnicos necessários para hospedagem, comunicação ou diagnóstico. Esses provedores devem tratar dados conforme a finalidade do serviço.

6. Seus direitos

Você pode solicitar acesso, correção ou exclusão de dados quando permitido pela operação do serviço e pela lei aplicável.

7. Contato

Para dúvidas de privacidade, solicite atendimento por e-mail em alphaxapp@gmail.com.
